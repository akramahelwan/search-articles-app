Search Articles App
Technologies Used:

React
JavaScript
CSS

How to Run the Project:

Clone the repository
git clone YOUR_REPOSITORY_URL
Navigate to the project folder
cd search-app
Install dependencies
npm install
Start the development server
npm start

The application will run at:

http://localhost:3000

Notes:

I used a local array of articles as requested in the assignment. The search functionality filters articles in real time and highlights matching text inside the title and content.
